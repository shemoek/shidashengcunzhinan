# 石大探店指南

## 好望角篇

任何店铺都可以先去美团和抖音搜索一下，对价格有个了解，一般也有线上优惠（感觉抖音把美团的生意抢走不少）。

### 楼外商铺

- **蜜雪冰城**：好望角入口外左侧，这没啥介绍的，虽然我很少喝饮品，但确实是人民的奶茶店。
- **华莱士**：在蜜雪冰城隔壁，楼内没有入口，从外侧进入，建议去抖音华莱士官方账号买，几乎是24h直播，价格更便宜，然后打开定位，直接借助华莱士的官方小程序提前线上点单，十分钟左右出餐，刚好到店，快餐都懂没什么好说的，价格又更实惠便宜，很难吃腻。

### 一楼

- **济宁肉夹馍**：入门左手的窗口，6-10元上下，我喜欢早上去买一份，然后去旁边的超意兴买一个茶叶蛋，一碗牛奶燕麦粥。
- **手机维修店**：入门右手，只来过一次，不过估计跟所有的电脑城和修手机店铺一样，多少都有点坑人的意思，网购过分发达便宜，线下不容易活，哪怕很良心的也感觉有点坑。<br>去贴柔性手机膜，网上个位数的价格就好几片，但他这里只有钢化膜，并且40一片，除了手机膜其他的不知道价格，但能做这么久，应该也不会太过分。
- **超意兴**：食堂类，推着托盘选餐，家常饭，种类也不少，不算美味，但价格实惠可控，连锁餐饮干净有基本保障，但是店里不知道是清洁剂还是什么，总有一股怪怪的味道，就像我不喜欢坐车闻空气清新剂一样。
- **鸡公煲**：18元，米饭免费续，去过几次，可以选干锅汤锅，还可以。
- **暖溢水饺**：专卖饺子的，干净整洁，自己调蘸碟，想吃饺子了就来这家。
- **土豆粉**：新开店有抖音优惠，真的挺辣的，吃的很痛苦，满头大汗，建议辣度尽量选低一点，可以两掺面食之类的。跟超意兴有个同样的问题，用的清洁剂有点怪味，不过最近淡了很多。
- **小土豆餐厅**：在普遍使用扫码点单的当下，保持了北方老式小店的气质，没有主打菜（烩面馆、鸡公煲这种卖特定产品）卖的很杂，也比较全，味道还行。
- **面馆**：主要就卖面，不错，大概是15元往上，干净整洁，像连锁店一样。

另外有两家只去过一次，麻辣香锅和冒菜，忘了，好像都挺辣，可以先线上搜一下。

### 二楼

- **水煮肉片**：自动扶梯怼脸，也是挺辣的，味道还行，只去过一次（好像不太能吃辣？）
- **理发店**：没去过，四年全在校内邮政旁边的理发店剪的，感觉很有必要专开一篇理发篇，但实在缺乏经验，关于校内理发店又很难评，很难昧着良心说他剪的满意。
  <br>但就二十年的人生经验而言，似乎就没有满意过，所以也不能说是他的问题，只能说中规中矩，每次理发之后的必然会有的心路历程，不必心存多余的幻想。
- **新村韩食**：去了很多很多次，就是韩国夫妻的小店，收拾的干净整洁印象很好，地道，有各种拌饭和汤，种类不多不少刚刚好，我比较喜欢点蜜汁烤肉（肉和洋葱）和一碗米饭，再免费续一碗，或者辛拉面/火鸡面加米饭。
  <br>蜜汁烤肉吃太多次了，导致吃腻了有点反胃，上次去的时候发现肉好像少了一些，一次性筷子也换成了干净的金属餐具，依然很整洁干净，希望老板也不要太追求降成本。
  <br>有免费的两种泡菜，一个是经典的辣白菜，那叫一个地道，味道不错，青岛离韩国比回家都近，所以韩餐比较多，但是那些连锁商业化的韩餐，辣白菜搞的感觉都不太地道，不伦不类的；另外一个小菜是淡黄色的，萝卜片？味道也不错。
  <br>比较偏爱这家店的原因是对韩餐好奇，外加比较有人(情)味，老板不会主动打扰，不过有时候会和家人或者主动搭话的客人聊天，他们家的孩子似乎也在国内上学所以来开的小店（不多透漏隐私了）。
- **披萨店**：二三十元吧，初去时发现外国人很多，蛮惊讶的。第一次见这么多外国人在校外，最开始还以为是因为披萨很地道，后来发现这不是主要原因（当然味道也不错）主要原因是一个极其简单的细节——他们家的小程序带英文，做商业开店有时候一点细节真的影响很多。
  <br>最普通的披萨是25元，普通的套餐是33元：一个披萨，一份薯条（这个炸的确实一般），一杯可乐。现烤的披萨，刚出炉的芝士拉丝，挺不错，这一套简单粗暴的搭配真的还挺有国外餐饮气质的。
  <br>不过外国人多这一点是优点还是缺点看个人喜好吧，外国人体味还真挺重的，不论男女。来留学的基本都是第三世界小国（没有任何看不起我油专的意思）根据报道，96%生源都是一带一路的沿线国家，主要生源地是中亚和非洲，还有阿拉伯、南亚，貌似还有墨西哥的，可以等学校国际周看看国旗，没几个认识的国家。
  <br>个别的体味轻，可能是更勤于洗澡的原因，外观就可以作为一定的参考。有些人穿着整洁，体味也就更轻，甚至没有味道，不知道是不是巧合，这些人的汉语经常意外的不错，交流时甚至有几分中式的客气，看得出来真的有在认真学习，自愧不如。
  <br>而有些拖着鞋或者脏衣服，味道可能更重一点，一股浓重的香水味混合体臭味。
  <br>经济发展的确和居民生活习惯高度相关，发展不太好的小国留学生，似乎就像n年前我国的县城乡镇一样，个人卫生注意的偏少一点。（实在不想回忆大一路过被熏愣神的初次体验）
- **忆香小厨**：焖面，20元上下，味道不错，要就口两蒜吃。
  <br>店家送的一碟小菜是用来解腻的，但小菜太辣了，而且味道不太喜欢，还是吃蒜更舒服。
  <br>面确实挺好吃的，自己取杯子，桌子上有柠檬水也很解腻。
- **爆爪**：只去过一次，已经忘了，生意相当不错，喜欢啃鸡爪的人应该会很喜欢，但我不喜欢啃鸡爪，把小菜当正菜不顶饿啊。
- **蹄花洋芋饭**：不错，名字就是特色，蹄花和洋芋饭，生意也不错，老板很友善。
  <br>但肠胃不好的建议吃普通米饭就行，洋芋饭很香，但总感觉有点腻腻的，多吃几口就有点不行了。
- **辣隐砂锅菜**：砂锅热菜，刚端上来非常烫，千万不要把手碰到砂锅上，堪比刚断电的电热板，经验之谈。
  <br>主要只吃过砂锅鸡，味道不错，老板娘很热情，经常去吃，米饭自己盛，免费续，送一瓶矿泉水或者一盒小甜水。
- **大盘鸡**：从外面往里看，门口有个笼子样式的栏杆，这个装修审美实在不敢恭维，感觉这一个栏杆拦住了不少客户（感觉是遮挡店外的视线，注意隐私的目的？不过客户都出来吃饭了，又不是来开房间，这个隐私保护实在不敢恭维）。
  <br>但进去之后发现意外的大，干净整净，也是很有老式餐厅的气质，味道也很不错，老板夫妻很友善，希望老板哪天把栏杆卸了，生意可能会好不少。
- **地锅鸡**：新开的店，抖音上17一份地锅鸡，味道还挺不错，四个小饼蘸汤吃，看样子应该是能活下去的。

### 三楼

几年前好望角又脏又破，完全没生意，这两年终于熬出头，逐渐变好。

但三楼也仅在前几年有几个外卖店铺，同样又脏又破，最近两年自动扶梯被堵住，好像一直在装修，不知道三楼什么情况。

前几天见到好像有人在三楼逛，有机会去看看。

### 四楼

坐电梯上去。

- **IM电竞网吧**：不错，大概5元/h上下，在前台可以使用身份证或者面部识别，然后直接去开机身份证号登录就行。
  <br>会员充值优惠，非会员很贵，但充一点就算会员，算是个留回头客的手段，可以在美团抖音上找劵用。
  <br>由于以前不知道网吧有啥意思，所以直到去年才第一次去网吧，又由于玩的游戏种类不多，第一次去依然不知道有啥意思，后来某日晚上睡不着觉第二次去，买了个5h的通宵（电竞舞台和电竞大厅居然不是一个位置，我在侧边的台子上打了半天）正巧那段时间开始玩三角洲，然后意外发现体验完全不同了，每一把都轻松刷140个人机脑袋天才少年，第一次感受到器械灵敏度和耳机的差距真的很大，之后才发现自己的电脑鼠标小桌面，玩的确实不太舒服。
  <br>网吧的作用就是和好友坐一起快乐打游戏，或者是在线竞技、fps枪战之类的对网速和器械灵敏度有要求的这类游戏，去网吧玩，体验的改善还是相当显著的。
  <br>网吧电脑的桌面上居然还配备租号代打之类的软件，一条龙服务相当完善。
- **24小时健身房**：24h健身房，还在同层扩张了二店，场地比较大，东西都比较齐全，新手老手都是不错的选择。
  <br>但当时误刷了两个劵，老板没退我钱，很不爽，再加上之后就去西海岸铁馆了（参照健身篇）所以没再去过，但确实是一个不错的选择，同学也多，没压力好交流。
- **公共厕所**：四楼有唯一的一个厕所，很小很窄。

### 好望角夜宵篇

一般是晚上9点左右到凌晨，在停车场外的路边就已经成一排了，等着9点之后好望角楼内商户大多闭店之后夜宵才能进入。

自从这两年好望角生意开始变好之后，就慢慢的开始有人在晚上在门口的区域摆摊，管理人员虽然一天到晚没看出来有啥用，扶梯八百年不修又经常坏，但收钱还是很敏锐的，及时的对夜晚摆摊的收取了费用，600rmb一个月，后面涨没涨价不清楚，反正去年刚开始那会跟一阿姨唠嗑是这情况。

夜宵的摆摊更替非常频繁，经常有新人出现又消失，比前两年楼内的商户倒闭还快，但固定npc还是有几位的，比如炒饭有三四个摊位，有牛杂，有烤肠面筋，有卖粥的，有肉夹馍的。商户之间可能有争斗，晚上摆摊估计也不容易。

去年有一个周边美容店的兼职摆摊，做了红烧肉鹌鹑蛋配调爽口凉菜，13r一份，刚买的时候懒得加微信不屑一顾，但吃了之后惊为天人，吃完后特地从学校扭头又跑回去悄悄添加了微信，暗暗成为红烧肉盖饭最虔诚的信徒，但可惜一周没撑住，惊艳了我整个冬天的红烧肉盖饭，成了惊鸿一瞥死去的白月光，我甚至因为连吃几天有点腻，傲慢的错过了它的最后一次出摊，没想到竟是永别，直到失去了才懂得珍惜。

感觉要么是商户争斗，要么是当时天太冷。不过根据之后的情况来看，也有可能是白天提前做一大桶饭太耽误正事，又要搬着大桶来回跑，干脆懒得来了，现实总是这么戏剧又残酷。

旁边荷悦里一条街貌似也有夜摊，但只去过一两次，后面就忘了，不知道现在还有没有。

### 好望角隔壁

西侧有栋小楼，卖拉面，小笼包，八点半烧烤餐厅，没吃过，拉面生意貌似还行，小笼包估计主要是早餐。

### 家家悦

很喜欢这个超市，自从中学之后就没怎么逛过超市了，在这里有点以前跟着家长一起逛大超市的感觉。

玩具区很小，卖的还有飞行棋大富翁，现在的小孩都不怎么玩了。

报手机号就能有个会员，每次买三五块钱东西都能攒积分，完全忘记有啥用。

卖的东西挺全的，最近5块钱顺手买了几根猫条喂狗，校猫神出鬼没的没找到。

虽然猫条是边角料做的很廉价，但实在太香了，狗都能吃人未必不可，问了AI是无毒的。不过貌似非常不健康，还很大可能会蹿，暂且保留此种可能，等哪天睡桥洞了也许会尝试猫条夹馍。

## 荷悦里篇

荷悦里在北门对面，离宿舍楼有点远，去的次数比较少，但确实比西北门这块漂亮不少，建的还挺气派。

- **德鑫排骨**：干净整洁，店面不小，味道也不错，谈不上很好吃，中规中矩的不错。
- **兰州牛肉面**：你是兰州牛肉面，再怎么讲故事它也只是兰州牛肉拉面，就是建成皇宫，天天讲朱元璋康熙爱吃的故事，它也只是兰州牛肉拉面。
  <br>15一碗，20元还能加个咸菜，贵的这点价钱全扔装修上了，一楼的小电梯挺有样的，二楼店面不大，弄的挺漂亮，有点日本小店那样，期待那两片肉也能是和牛。
- **馋笼大虾包**：连锁店，真海鲜啊？现做10分钟多点，味道…我感觉海鲜和包子它压根就不搭，看个人喜好。

其他店铺没怎么去过，荷悦里这几家店的气质就是更精致一点，流水线一样的干净整洁，但缺乏特色，不如西北门好望角那小店一家各有一家的味道，特点鲜明。

## 康大篇

我一直以为康大就是纯宾馆，后来才知道里面是店铺，大城市的玩法就是不一样。但又不是商场，这让人怎么知道店铺？纯互联网推广吗？

鲨鱼健身房、KTV、台球，楼内没怎么去过，只在抖音上刷到一家拼豆店，有点寒酸，但拼豆也确实不需要啥（感觉就是小时候公园的烤胶画啊，过两年换个壳还是当年那一批人玩...）。

康大楼后一条街堪比阴阳街，前两年都不知道那后面还有东西。

- **兰州牛肉面**：味道没啥说的，比较干净整洁，价格也不贵，抖音美团上都有店铺，可以先看一下。
- **新疆抓饭——巴郎子**：大门挺高挺沉的不好打开，店面积比较小，外卖比较多一点。
- **新疆抓饭——疆道**：店面要大一些，也收拾的更干净整洁，生意也不错，抓饭有点油，不算很腻。
  <br>新疆饭还是挺有意思的，烤馕包子抓饭羊排什么的都是硬菜，新疆人新疆味（艾热shark那个歌真不错。）。
- **沙县小吃**：估计是主打外卖的，店内小，有点乱，也没多少座位。

## 城发悦邻广场篇

去西铁每天都能看到，就顺路去逛了一下。

首先二楼厕所很赞。

- **一楼广顺兴**：店蛮大的，广式餐厅哦，看着就很有气势哦。于是在月末余额只剩五十元的时候洒脱的走了进去，光棍一条，就这五十，花光正好，多了没有。（结果还剩了十块）
  <br>没点汤没点菜，就点了个豆豉鲮鱼煲仔饭、一个叉烧包、一个冰冻芒果。服务员态度倒是蛮不错的，还帮我把饭拌好，煲仔饭很好吃，很正宗，不过自己吃一大份有一点点腻。
  <br>叉烧包有两点很惊讶，第一点是竟然只有一个（还以为类似小笼包），还是切开四瓣的，第二点是竟然是甜的，又增加了一点腻味。
  <br>芒果解腻，可惜芒果这么好吃的水果冰冻之后也索然无味。
  <br>单人去吃确实很难吃完，哪怕点的东西很少，多人去吃的话估计就会很实惠，也可以多点两个汤菜。

一楼其他还有好几家餐厅，生意好像都蛮不错的，人流量都比较大，干净整洁，虽说不是面积非常大的商场。

## 剧本杀 & 桌游

- **M7推理社**：应该是桌游剧本杀类型的店。大学之后对于此类游戏其实相当有兴趣，能和朋友一起玩，又能真人代入各种情节故事，所有人一起配合，实在有趣。

此类重剧情的桌游类型，大致源自美国，本来的玩法是带DM的桌上角色扮演游戏，有走棋的元素，几个玩家各自选一个身份，一位主持人投骰子，用纸笔写一些东西(小学挺爱玩类似的游戏)，这些年极其出名的战锤40k，老辈子喜欢的dnd龙与地下城都是这种类型的。

这些桌游的玩法在国内被本土化，剧本杀保留了DM，弱化甚至取消了桌游部分的内容，强化了玩家间的语言交流，剧情的内容变得更多，但不怎么玩游戏了，所以陌生人之间缺乏交流，就不如朋友一起去玩的开心。

我只玩过一次剧本杀，店家叫异世馆（当时在某大楼里租的几间屋子有点寒酸，现在貌似已经有钱搬家了）工作人员都很认真负责，98元的价格，不便宜，还有不少更贵的，不过时间很长，从下午玩到晚上，偶尔体验一下挺好，实在没钱去多玩几次，而且不是和朋友一起体验大打折扣，不过人家工作人员也确实陪你玩了那么久，玩的也确实挺不错的，很开心。

当时由于过于认真想赢（实际剧本杀没有任何输赢设计）忘记要多带入剧情了，在那琢磨推进度，关键这剧本作者写的也不行，搬运的什么狗屁案件情节，小时候看少年包青天就看过，估计看过那一集的全中国的观众都知道答案，小嘴一张就直接把答案说出来了，别人正打算扮演凶手深度参与呢被点了，有点丧失体验感，去玩剧本杀的小伙伴一定注意，剧本杀没有赢不赢一说，跟狼人杀不一样，好好体验就好。

关于桌游（不算扑克象棋围棋）了解的最早的是龙与地下城的ip，在学校图书馆一楼文艺作品区（回头要写个图书馆篇）在读奇幻类小说的时候看到了《龙枪》和《龙枪编年史》系列（一楼还有很多科幻小说，各种国外经典IP作品，真看饱了）算是衍生小说，读着也还行，在上个世纪可能确实相当新奇，但对于已经观看过太多故事的现代人来说，已经不够精彩了。作者文笔一般，但想象力丰富，很多后来的奇幻故事概念和角色都发源于此，甚至被用烂了，这也是这个小说读起来已经没那么惊艳的原因之一，相较于魔戒来说感觉节奏太快了。

也许这种史诗奇幻就适合像魔戒一样，由一个超级慢节奏的语言学教授老登兼职写着玩，一会又诗歌一会又语言学的，极其缓慢的建立一种史诗的氛围，文笔又不错，于是一边被慢节奏折磨一边耐着性子冒险。

这些桌游好像都是有专业的公司专门编写衍生故事，设计各种相关游戏和ip内容。

学校有桌游社，桌游项目非常丰富，强烈建议和室友朋友一起玩桌游，非常有趣，不需要很难，玩简单的桌游甚至扑克都可以，没什么成本在一起玩的非常快乐，也不会连跪红温。

桌游行业也比想象的更发达，似乎是国外青少年玩家比较多，可以参考很多欧美剧作品，里面的确有很多此类元素，比较出名的有怪奇物语，勇敢者游戏，大都是青少年幻想自己进行一场盛大的奇幻冒险，就像日本高中生拯救世界一样。

这种重剧情冲突，重主题的是美式桌游，还有德式桌游，由于两次世界大战的经历，游戏内容轻冲突，重策略。

我大中华最出名的首部桌游是三国杀，国货之光，好评如潮。

光谷周边其余店铺也很多，但有点远了而且天黑了，懒得再逛了。

# 石大生存指南（体育篇）

最近阅读了《银河系搭车客指南》之后，突然再次燃起了写一份石大生存指南的欲望，虽然两者没有什么关系，当然这本书还是很好看的。

希望能有一点银河系搭车客指南的幽默，以及游戏解锁新地图一样有趣的气质。

我前两年几乎没走出过校门，这多少有点遗憾，其实就三五分钟的步程，就能大大改善大学生活质量。

我觉得“信息差”这个词被各种焦虑营销有点用坏了，其实就跟我突然发现荟萃食堂二楼的猪脚饭挺好吃一样，吃猪脚饭没有难度，这也不是什么难获取的信息，但有时候你就是不知道。

大学四年时间，我依然有几个食堂窗口还没去吃过，这个指南的用处就是给大家省点事，不用一个个窗口吃过去就能有个参考，提前点亮新地图给大家降低探索难度。

从互联网到AI，这种“信息差”类问题越来越少，给大家增添一点帮助。

由于我在舒适区待的时间比较久，前期很多区域不敢探索，走过的地方还是比较少，很多地图和商店都没解锁，如果同学们有去过的，可以补充一下。
